# Child Cost Estimator

The **Child Cost Estimator** is a web-based application designed to provide an estimate of the annual cost of raising a child based on user inputs such as location, annual household income, and family size. The application features a clean and modern user interface with dynamic result display.

## Features

- **Responsive UI**: Clean and modern design that works well on both desktop and mobile devices.
- **Form Validation**: Ensures all required fields are filled out correctly.
- **Dynamic Calculation**: Provides an estimated cost without reloading the page.

## Technologies Used

- **HTML**: For the structure of the web page.
- **CSS**: For styling the user interface.
- **JavaScript**: For form validation and dynamic result display.

## Usage

1. Clone the repository:
    ```sh
    git clone https://github.com/your-username/child-cost-estimator.git
    ```
2. Navigate to the project directory:
    ```sh
    cd child-cost-estimator
    ```
3. Open `index.html` in your preferred web browser.

## Project Structure

```
child-cost-estimator/
├── index.html
├── style.css
└── script.js
```

- `index.html`: The main HTML file containing the structure of the web page.
- `style.css`: The CSS file for styling the web page.
- `script.js`: The JavaScript file for form validation and dynamic result display.

## Form Fields

- **Location**: Enter your city or region.
- **Annual Household Income**: Enter your annual income.
- **Family Size**: Enter the number of people in your household.
